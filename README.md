# ESHC_scanner
Описание:

Данное приложение позволяет контролировать состояние сотрудников на различных объектах. В определенное время (7 раз в сутки) сотрудник должен совершить "отзвон" в колл-центр (посредством считывание QR кода). В случае удачного "отзвона" оператор колл-центра понимает, что на объекте условно "все хорошо" и знает Ф.И.О сотрудника. В случае отсутствия "отзвона" оператор колл-центра самостоятельно связывается с сотрудником на объекте. На стороне оператора колл-центра установлено приложение отображающее список всех объектов (70 шт.), которые должны совершить "отзвон".
